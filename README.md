# Personal Portfolio Website

## Overview
A Full Stack Personal Portfolio Website developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB Atlas. The application features a secure Admin Panel for managing portfolio information, projects, profile details, and contact messages through an intuitive dashboard.

---

## Features
- Responsive Portfolio Website
- Secure Admin Login with JWT Authentication
- Dashboard Overview
- Add, Edit & Delete Projects
- Portfolio Content Management
- Skills Management
- Contact Message Management
- Profile Management
- Profile Picture Upload
- Resume Download
- Mobile Responsive Design

## Installation

### Clone the Repository

```bash
git clone https://github.com/janishasweety1912-stack/Personal-Portfolio-Website.git
```

### Install Backend Dependencies
```bash
cd server
npm install
```

### Start the Backend
```bash
npm start
```
